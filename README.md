# 📓 Retro Daily Planner (tracker_proto)

A retro-themed, Notion-like daily planner and life tracker prototype designed to work offline-first. This application is built with vanilla HTML, CSS, and JavaScript, focusing on a robust, local-first data storage approach.

## ✨ Features

- **Retro Notion-style Interface**: A visually appealing sidebar layout with colored icons and a clean, retro aesthetic.
- **Calendar & Events**: Track your daily schedule with support for recurring events.
- **Task Management (To-Dos)**: Stay on top of your tasks and daily checklist.
- **Day Logs & Notes Workspace**: A Notion-style database table for detailed logging and a flexible layout system for notes.
- **Platform integrations**: Track coding platforms, health metrics, and suggest improvements.
- **Offline First**: All data is stored locally, so you can manage your day without an active internet connection!

## 🚀 Getting Started

Simply open the `index.html` file in your browser, or run a local development server in the project directory:

```bash
python -m http.server 8080
```
Then visit `http://localhost:8080` in your browser.
